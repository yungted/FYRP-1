# Sem 2 2025 FYRP-1
**Intuitive Human-Robot Interaction for Service Applications**
In this project, an intuitive HRI interface for a front-of-house greeting robot (showcase robot) will be developed. It utilizes LLMs and AIs from different sources, all integrated together through API calls. 

**Techniques used:**
1. MCP (Manual Context Protocol)
2. MQTT Broker for Communication between Raspberry Pi 5 and the Nvidia Jetson Nano
3. Automatic Speech Recognition and Text-to-speech for Intuitive Interaction


**APIs used**
ASR - https://deepgram.com/
Scene Perception - https://moondream.ai/
LLM - https://ollama.com/ (Model: GPT-120b)
